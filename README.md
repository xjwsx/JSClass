## JavaScript란?

JavaScript는 **웹 브라우저에서 동작하는 프로그래밍 언어**입니다.

### HTML, CSS, JavaScript의 역할

```
┌─────────────────────────────────────────────────┐
│                    웹 페이지                      │
├─────────────────────────────────────────────────┤
│  HTML      →  구조 (뼈대)      →  "무엇을 보여줄까?"  │
│  CSS       →  스타일 (디자인)  →  "어떻게 보여줄까?"  │
│  JavaScript →  동작 (기능)     →  "어떻게 반응할까?"  │
└─────────────────────────────────────────────────┘
```

### 예시

- **HTML**: "버튼이 있다"
- **CSS**: "버튼은 파란색이고 둥글다"
- **JavaScript**: "버튼을 클릭하면 알림이 뜬다"

## JavaScript의 특징

### 1. 인터프리터 언어

- 코드를 한 줄씩 실행합니다
- 별도의 컴파일 과정이 필요 없습니다

### 2. 동적 타입 언어

- 변수의 타입을 미리 지정하지 않아도 됩니다
- 실행 중에 타입이 자동으로 결정됩니다

### 3. 객체 기반 언어

- 모든 것이 객체로 구성됩니다

### 4. 이벤트 기반 프로그래밍

- 사용자의 클릭, 입력 등에 반응합니다

## 실습 방법

1. `.html` 파일을 웹 브라우저(Chrome 권장)로 엽니다
2. `F12` 키를 눌러 개발자 도구를 엽니다
3. `Console` 탭에서 결과를 확인합니다
4. 코드를 수정하고 새로고침(`F5`)하여 변화를 확인합니다
